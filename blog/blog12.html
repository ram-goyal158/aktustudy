⚡ Top 5 Mini Project Ideas for Electrical Engineering (AKTU)

Practical – Budget-Friendly – Syllabus-Based (2025)

🔌 1. Automatic Street Light Using LDR

Description: Lights turn ON/OFF automatically based on sunlight using LDR (Light Dependent Resistor).

🔧 Components Required:

  • LDR Sensor
  • Relay Module
  • Resistor, LED, 9V Battery
  • Transistor (BC547)

💡 Key Learning:

  • Sensor interfacing
  • Relay operation
  • Power-saving automation

🔋 2. Battery Level Indicator Using LEDs

Description: Indicates battery charge level using LED indicators like mobile battery bars.

🔧 Components Required:

  • 5 LEDs, Resistors
  • LM3914 / LM339 Comparator IC
  • 9V / 12V Battery

💡 Key Learning:

  • Analog signal reading
  • Comparator circuit logic
  • Indicator circuit design

🌞 3. Solar Mobile Charger Circuit

Description: Build a mobile charging setup using solar panel and voltage regulator.

🔧 Components Required:

  • 6V or 12V Solar Panel
  • IC 7805 (Voltage Regulator)
  • Diode, Resistor, USB Port

💡 Key Learning:

  • Solar energy & renewable basics
  • Voltage regulation
  • Power electronics

🌀 4. Speed Control of DC Motor Using PWM

Description: Control DC motor speed using PWM signal generated via potentiometer or Arduino.

🔧 Components Required:

  • DC Motor
  • MOSFET / Transistor
  • Potentiometer
  • Arduino UNO (optional)

💡 Key Learning:

  • PWM technique
  • Motor speed control
  • Analog to digital interfacing

🔋 5. Overvoltage and Undervoltage Protection System

Description: Protect appliances from voltage fluctuation by auto cut-off using comparator logic.

🔧 Components Required:

  • Relay
  • Voltage Comparator IC (LM324)
  • Zener Diodes
  • Transformer (optional)

💡 Key Learning:

  • Voltage sensing & control
  • Relay cutoff logic
  • Appliance protection

✅ Tips for a Successful Mini Project

  • Draw a clean & labeled circuit diagram
  • Explain your circuit confidently during viva
  • Maintain a project file with abstract, design, components list, observations, and result
