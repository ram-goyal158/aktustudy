🤖📒🎨 ChatGPT + Notion + Canva Guide for AKTU Students (2025)

Study smarter, not harder – master these 3 tools for academic success

🧠 1. ChatGPT – AI Study Assistant

🔗 Visit ChatGPT

  • Understand subjects like DBMS, Signals, or OOPs
  • Get code help for Python, Java, C++, HTML
  • Write seminar scripts, SOPs, resumes
  • Generate ideas for mini projects
✏️ Try:
• “Explain normalization in DBMS with examples.”
• “Suggest a mini project idea in IoT using Arduino.”
• “Draft a 2-min script to present on AI.”

📒 2. Notion – Note-taking + Project Management

🔗 Visit Notion

  • Create semester-wise subject notes
  • Track assignments, exams, and submissions
  • Use Kanban boards for mini-project milestones
  • Store ChatGPT responses or YouTube resources
🧩 Use Templates:
• Semester Dashboard – for notes, syllabus, resources
• Project Tracker – track tasks, deadlines, and reports
• Revision Planner – to revise subject-wise before exams

🎨 3. Canva – Create Presentations & Reports

🔗 Visit Canva

  • Create stunning presentations and report covers
  • Use templates for certificates, posters, charts
  • Design club invites or seminar infographics
  • Export to PDF, JPG, or PPT easily
💡 Pro Tip:
Start with “Presentation 16:9” or “A4 Report Cover” for academic work.

🧩 How to Use All Three in a Mini Project

  • ChatGPT: Understand topic + get code (e.g., weather app using API)
  • Notion: Plan project modules, write documentation, track progress
  • Canva: Make final report design & slides for viva presentation

🔁 Combine = no chaos + more marks + better teamwork

Start using all 3 in small ways — you’ll save time, study better, and shine in presentations.
