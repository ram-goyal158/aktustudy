📝 How to Build a Resume for AKTU Campus Placement – 2025 Guide

Your first impression starts with a great resume!

📌 Why Is Your Resume So Important?

  • ✅ First thing recruiters see before shortlisting
  • ✅ Reflects your education, skills & achievements
  • ✅ Must pass ATS (Applicant Tracking System)
  • ✅ A good format = more interview calls

🧱 Ideal Resume Format (AKTU Students – 1 Page Only)

1️⃣ Header / Personal Details

  • Full Name, Mobile No, Email ID
  • LinkedIn/GitHub (if applicable)
  • City & State (No full address/photo)

2️⃣ Career Objective (Optional)

“To obtain a position in a growth-oriented company where I can apply my technical knowledge, problem-solving ability, and communication skills...”

3️⃣ Education

DegreeInstituteYearCGPA/%

4️⃣ Technical Skills

  • Programming: C, C++, Java, Python
  • Web: HTML, CSS, JavaScript
  • Tools: Excel, AutoCAD, MATLAB
  • Domain: DBMS, OS, DSA, Networking

5️⃣ Projects

Smart Irrigation System (2023)
Built using Arduino + Soil Sensors + Relay Module.
GitHub: github.com/yourproject

6️⃣ Internships / Trainings

  • Web Dev Intern – Internshala (Remote)
    Built landing pages using HTML/CSS/JS

7️⃣ Certifications

  • Python for Everybody – Coursera
  • AWS Cloud Practitioner – AWS
  • Digital Marketing Basics – Google

8️⃣ Achievements

  • 🥇 1st in Hackathon, College Fest
  • 🧠 GATE 2024 Qualified
  • 📣 Tech Club Coordinator, XYZ Institute

9️⃣ Hobbies (Optional)

  • Blogging, Photography, Cricket, Tech News

❗ Resume Do’s & Don’ts

✅ Do:

  • Use clean 1-column layout
  • Save as PDF
  • Proofread grammar & spacing

❌ Don’t:

  • No photos or flashy colors
  • Don't lie or exaggerate skills
  • Don’t write “RESUME” on top

💡 Tools & Bonus Tips

  • Canva / NovoResume – For fast design
  • Overleaf – For LaTeX professional resumes
  • GitHub – Showcase tech projects
  • LinkedIn – Keep your profile updated

🎯 Final Thoughts

Start building your resume in 3rd year itself. Keep it updated every semester. Use this structure, focus on your strengths, and ask for reviews from teachers or seniors.
